# Метапромт: WhatsApp-раскладка композера + Google Drive медиа-мост (утв. 14.07.2026)

## Часть A. Раскладка значков композера как в WhatsApp
- «активный» = 24ч-окно ОТКРЫТО (проп windowOpen). Меняем ТОЛЬКО расположение/показ значков, механику скрытия клавиатуры не трогаем.
- Расположение (+, эмодзи, поле, микрофон/отправка) как в WhatsApp.
- Окно ОТКРЫТО: обычный ввод; 👋 и системные шаблоны СКРЫТЫ.
- Окно ЗАКРЫТО: свободный текст недоступен; показываем значок системных шаблонов (и 👋 как сейчас); клавиатуру НЕ показываем.
- Тап по значку шаблонов (закрытое окно) → попап со списком, полное содержимое каждого.
  - Без изменяемых частей → тап сразу ОТПРАВЛЯЕТ в чат.
  - С изменяемой частью (Schedule update → новое время) → выбор времени из доступных слотов класса (ноль ручного ввода) → отправка через admin_message с подставленным значением.

## Часть B. «+» → Google Drive медиа-мост
- Тренер «+» → фото/видео → сервер грузит ОРИГИНАЛ (без сжатия) на Drive hatabishchip@gmail.com в под-папку клиента (Gravity Stretching Media / <клиент>).
- Клиенту уходит ССЫЛКА (окно открыто - текст; закрыто - admin_message), не файл. В инбоксе у тренера видна ссылка.
- Клиент по ссылке видит только свою под-папку. scope drive.file, resumable upload, mapping клиент→folderId в БД.

## Ограничения
- Клиентские тексты - БЫЛО→СТАНЕТ перед изменением; без длинного тире.
- В основном веб; native-правки «+» → сборка 1.0.7.
