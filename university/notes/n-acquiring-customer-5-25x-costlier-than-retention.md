# Acquiring a new customer costs 5–25x more than retaining an existing one

Индустриальная оценка (Reichheld / Bain, агрегировано HBR): стоимость привлечения нового клиента в 5–25 раз выше стоимости удержания существующего. Широкий диапазон отражает разброс по индустриям и каналам, поэтому число — ориентир, а не константа. Для велнес-студии с дорогим локальным трафиком (Бали, платный CAC) верхняя часть диапазона правдоподобна. Confidence M: цифра вторичная, без выборки по студийному фитнесу.
