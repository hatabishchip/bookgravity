# Канон нейминга: Gravity Stretching

**Правило (владелец, 11.06.2026):** метод и бренд во ВСЕХ материалах называются **Gravity Stretching**.

## Запрещено
- "assisted stretching", "assisted-stretching studio"
- «ассистированная растяжка», «асистент стретчинг»
- "gravity-assisted" (полуформа — тоже нет)

## Разрешённые формулировки
- Gravity Stretching / gravity stretching (метод)
- spinal decompression in suspension straps / by hanging
- декомпрессия позвоночника в подвисе / в стропах

## Почему
"Assisted stretching" — категория конкурентов (Stretchr — «first assisted stretching brand in Indonesia», StretchLab — глобальная франшиза). Использовать их термин = играть на их поле и размывать бренд. Gravity Stretching — собственная категория (терапевтическая инверсия), бренд = имя метода.

## Исключения (единственные)
1. Verbatim-цитаты клиентов в research/.
2. Фактические описания конкурентов в research/ (их официальные самоназвания).
3. Код bookgravity: `assistedSlots` / "as assistant" = тренер-ассистент на классе (другое понятие).

## Чек перед публикацией
`grep -ri "assisted" <файлы>` — должно быть пусто (вне исключений).

# Дополнение (11.06.2026): НЕ аэро-йога, НЕ гамаки — разница метода

**Приказ владельца:** в текстах также запрещены «aerial yoga», «aero yoga», «аэро-йога», «fly yoga», «hammock», «гамак» как описание НАШЕГО метода. Единственное допустимое употребление — мифоборчество с явным отрицанием («No fabric hammocks here — individual suspension straps») и фактические описания конкурентов (Udara, FlyHigh = аэро-йога).

## Чем Gravity Stretching отличается (изучено на gravity-stretching.com, метод Андрея Молодцова)

| | Gravity Stretching | Аэро-йога / fly yoga |
|---|---|---|
| Снаряжение | **Отдельные стропы/верёвки** («верёвочки») с ручками-карабинами | Тканевый гамак-кокон |
| Поддержка | Точки контакта: держишься руками/ногами за стропы; стропы держат 100% веса | Ткань обволакивает всё тело |
| Цель | **Система оздоровления**: терапевтическая декомпрессия позвоночника, «гравитация мягко вытягивает тело - оно оживает и восстанавливается» (было «исцеляя» - язык чуда, заменено 24.07 по словарю Андрея) | Акробатика, перформанс, развлечение |
| Принцип | Работа «на расслаблении», без усилия; «используем гравитацию нам на благо» | Активные связки и позы |
| Структура | 7 уровней от простого к сложному (5 доступны любому - канон Андрея, исправлено 24.07), безопасная прогрессия для любого возраста | Класс-перформанс |

# Дополнение (24.07.2026): канон Андрея - «лианы», НЕ «в стропах» (перебивает список выше)

**Правило (владелец + канон andrey-voice 02.07, применено системно 24.07):** конструкции
«spinal decompression in suspension straps», «декомпрессия в стропах», «подвис в стропах»,
«di tali suspensi» - ЗАПРЕЩЕНЫ во всех НОВЫХ текстах на всех языках («насильно умная фраза,
не воспринимаемая»). Строки «Разрешённые формулировки» и «Словарь» выше в части строп -
устарели, действует этот раздел.

## Канон по языкам
- EN: «a gentle hang on the lianas - the body decompresses, space comes back to the spine,
  pressure comes off». Слово подвеса - только «lianas».
- RU: «мягкий подвис тела на лианах - декомпрессия создаёт пространство и снимает давление»
  («подвис» - существительное, как говорит Андрей; глагол «подвисает» не использовать).
- BH: «tubuh bergantung lembut di liana - tulang belakang mendapat ruang, tekanan berkurang».

## Словарь оборудования (канон)
- «lianas»/«лианы» - ЕДИНСТВЕННОЕ слово про подвес (владелец 24.07: только «лианы», без «верёвок» даже в скобках);
- «straps»/«стропы» - ТОЛЬКО опора для ног;
- «loops»/«петли» - пальцы рук/ног;
- Мифоборчество теперь: «No fabric hammocks here - individual lianas».

## Границы
- Живые объявления-победители (pain1 c «suspension straps») НЕ переписывать - смена текста
  сбрасывает обучение аукциона. Канон обязателен для всех НОВЫХ текстов: реклама, портал,
  посты, ответы агента, лендинги.
- Чек перед публикацией: grep -riE "suspension strap|в стропах|di tali suspensi" <файлы> -
  пусто вне цитат/описаний конкурентов.
