# Escalation queue

Вопросы/решения, где Claude не может (или не должен) принять решение сам. Пользователь должен ответить или разрешить.

Формат:

```
## <YYYY-MM-DD HH:MM>: краткая суть
Context: что произошло / какой источник / какой claim
Question: конкретный вопрос к пользователю
Status: open | resolved
Resolution: (заполняется когда ответ получен)
```
